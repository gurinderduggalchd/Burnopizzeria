FEATURES IMPLEMENTED


  • Home page shows the introduction, history, menu items, address, opening hours
  • User registration form have all the validation builtin for all the data entered
  • No field can be left empty. Post code is 4 characters long
  • Email address should be in correct format. Phone number should be 10 digits long only
  • Password should be at least 8 character long
  • In order page, select the pizza type and pizza price is displayed automatically
  • When delivery type is set to delivery, the text box for delivery address get enabled
  • When checkbox for "same as delivery" is checked, it checks the delivery address and display the delivery address in billing address textbox
  • Select the pay on to be online it displays the drop down list for credit card and textbox for credit card number
  • when credit card is visa or mastercard, the credit card number is 16 digits long and when credit card is american express, credit card number length is 15 digits long
